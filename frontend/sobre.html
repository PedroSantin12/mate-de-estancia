Nossa proposta

Uma loja gaúcha pensada para quem valoriza o ritual do mate.

A Mate de Estância é uma simulação de e-commerce criada para demonstrar uma aplicação web completa, com front-end, API REST, autenticação, carrinho, checkout e painel administrativo.

Tradição

O catálogo valoriza produtos ligados ao chimarrão: ervas, cuias, bombas, térmicas e kits para diferentes estilos de mate.

Tecnologia

A loja usa HTML, CSS e JavaScript no front-end, enquanto o back-end em Node.js expõe rotas REST documentadas com Swagger/OpenAPI.

Experiência

O cliente pode navegar livremente, salvar carrinho, calcular frete por CEP e finalizar pedidos após autenticação.
